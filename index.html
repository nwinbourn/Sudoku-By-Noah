Sudoku

by Noah Winbourn

Sudoku By Noah is a browser Sudoku game I built as a school project.

Controls

  • Click a cell or use WASD to move.
  • Delete: delete / backspace / 0 to clear.
  • Hold Shift and drag / move to highlight cells.
  • Press E or use Pencil Mode
  • Help Mode will validate inputs

How to Play Sudoku

  • Each row must contain the digits 1 through 9 with no repeats.
  • Each column must contain the digits 1 through 9 with no repeats.
  • Each 3×3 box must contain the digits 1 through 9 with no repeats.
  • Given numbers cannot be changed.
  • Fill in the blank cells so that all these rules are satisfied.
